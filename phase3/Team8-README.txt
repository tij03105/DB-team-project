-실행 및 사용방법
 압축한 자바파일을 해제한 뒤 이클립스에서 프로젝트로 phase3_team8을 추가한다.
 해당 프로젝트의 libraries에 ojdbc7.jar를 추가한다.
 코드를 실행후 콘솔창의 입출력으로 아래 서술할 기능들을 활용한다.

 
-기능
프로그램 실행시 비 로그인 상태이므로, 회원가입과 로그인 기능을 선택할 수 있다.
회원가입 기능을 선택하면 1입력후 엔터, 로그인 기능을 선택하려면 2입력후 엔터를 누른다.

1. 회원가입 기능 선택시(필수기능 1-A)
*가 붙은 정보는 생략할 수 없는 필수 정보이므로 반드시 입력한다.
ID는 15자리 이하의 값을 입력한다.
공백 및 15자리를 초과하는 경우 사용자에게 다시 입력을 받는다.
필수적으로 ID는 Account의 key값이므로 DB를 확인하여 중복된 값일 경우 사용자에게 알려준다.
password는 20자리 이하의 값을 입력한다.
공백 및 20자리를 초과하는 경우 사용자에게 다시 입력을 받는다.
phone number는 XXX-XXXX-XXXX 형식으로 입력한다.
공백 및 숫자가 아닌값 또는 형식에 맞지 않을 시 다시 입력을 받는다.
name는 20자리 이하의 값을 입력한다.
공백 및 20자리를 초과하는 경우 사용자에게 다시 입력을 받는다.
다음으로 address를 입력한다.
address의 경우 필수정보가 아니므로 생략이 가능하다.
sex는 0 또는 1을 입력하거나 엔터를 눌러 생략이 가능하고 이외의 값의 경우 다시 입력을 받는다.
생년월일은 YYYYMMDD의 형식으로 입력을 한다.
숫자가 아닌 값 또는 입력형식이 다른 경우에 다시 입력을 받는다.
다음으로 직업을 입력한다.
직업은 필수정보가 아니므로 생략이 가능하다.
모든 계정은 생성시 기본적으로 Basic 멤버쉽 상태로 생성된다.
입력한 정보를 바탕으로 생략한 정보는 NULL를 사용하여 DB에 튜플을 생성한다.

2. 로그인 기능 선택시(필수기능 1-D)
ID와 password를 입력한다. 공백 입력시 다시 입력을 받는다.
DB를 확인한 후 일치하는 튜플이 있으면 로그인에 성공한다.
일치하는 튜플이 없는경우에 존재하지 않는다고 알려주고 비 로그인 메뉴로 돌아간다.
관리자 계정으로 로그인을 하면 사용자에게 알려준다.

로그인에 성공하면 관리자 계정 또는 회원 계정에 따라 보여주는 메뉴가 다르다.
메뉴에 각 기능을 선택하려면 해당하는 숫자를 입력하면 된다.
1. 회원정보 수정 기능(필수기능 1-B)
수정하고자 하는 정보에 대해서 입력을 한다. 수정을 원하지 않는 정보는 공백을 입력한다.
각 입력에 회원가입과 마찬가지로 입력형식을 체크한다.
입력한 정보를 바탕으로 DB에 있는 튜플을 업데이트한다.

2. 비밀번호 수정 기능(필수기능 1-C)
변경할 비밀번호를 입력하고, 입력한 비밀번호를 한번 더 입력한다.

3. 회원탈퇴 기능(필수기능 1-D)
회원탈퇴를 선택하면 유저에게 정말로 회원탈퇴를 할것인지 다시 물어본후
y, Y, yes, YES를 입력하면 DB에서 해당 계정의 튜플을 삭제한다.

4. 로그아웃 기능
선택시 비 로그인 메뉴로 돌아간다.

5. 평가한 영상 보기(필수 기능 3-B)
자신이 평가한 모든 영상의 정보를 볼 수 있다. 이 때 제공되는 정보는 영상물의 제목과 평균 평점, 본인이 내린 평점이다. 

아래의 영상검색 기능(6,7,8)은 검색 후 영상물의 정보 확인과 더불어 평가여부를 묻는 기능까지 모두 포함한다 (필수 기능 2-D)
아래의 영상검색 기능(6,7,8)은 회원이 평가한 영상물을 제외하는 기능을 모두 포함한다((필수 기능 2-E)
6. 모든 영상 검색 기능(필수 기능 2-A)
시스템에 등록된 모든 영상물 중에서 사용자가 평가하지 않은 영상물들을 나열한다. 각 행은 영상물의 고유식별번호(TCONST)와 제목(TITLE)을 포함한다. 이후 사용자에게 정보 확인 및 평가할 영상물의 식별번호를 입력받는다.(ex t00000216). 입력한 문자열과 같은 식별번호가 없다면 검색결과가 없음을 알리고 재입력할지 선택받는다. 거절하면 메뉴화면으로 돌아간다. 
정보 확인 및 평가할 영상물의 식별번호를 정확히 입력하면 영상물의 정보가 나열된다.
ex)	식별번호 : t00000216
	영상물 제목 : Bored
	영상물 타입 : knuMovieDB_Original
	영상물 상영연도 및 날짜 : 2020-09-16
	상영시간(분) : 183
	영상물 장르 : Drama
	영상물 버젼 종류 : SP KR CN RU UA JP IN AU 
	영상물 평균 평점 : 7.5 
이후 이 영상물에 대해서 평가를 진행할것인지 요청받고 거절하면 메뉴로 돌아간다.
평가를 선택하면 영상물에 대한 평점을 0.0에서 10.0점 사이의 실수로 입력받는다. 조건에 맞지 않는 형식은 재입력을 요청한다. 입력이 완료되면 평가 입력이 완료되었다고 알리며 메뉴로 돌아간다.

7. 제목으로 영상 검색 기능(필수 기능 2-B)
검색할 제목을 입력받는다. 문자열을 입력하면 제목에 해당 문자열이 포함된 영상물들의 식별번호와 제목을 나열한다. 검색결과가 없으면 결과없음을 출력하고 메뉴로 돌아간다. 
검색결과가 출력되면 6번에서와 같이 사용자에게 정보 확인 및 평가할 영상물의 식별번호를 입력받는다.(ex t00000216). 이후 평가방식은 동일하다.

8. 조건으로 영상 검색 기능(필수 기능 2-C)
영상물을 검색하기 위한 조건을 순차적으로 입력받는다. 
 8-1. 영상물 버전을 선택한다. 버전을 선택하지 않으면 공백을 입력하여 오리지날 버전이 선택된다.
사용자는 숫자메뉴를 하나씩 입력하여 버전을 선택할 수 있다. 하나씩 선택될 때마다 버전메뉴에서 선택된 버전은 삭제되고 다시 목록을 나열한다. 여러개의 버전을 선택할 수 있으며 선택을 종료시에 공백을 입력한다.
 8-2. 영상물 타입을 선택한다. 상관없다면 0을 입력한다.
사용자는 숫자메뉴를 하나 입력하여 영상물 타입을 선택할 수 있다. 하나만 선택할 수 있다. 
 8-3. 영상물 장르를 선택한다. 장르가 상관없다면 공백을 입력한다. 
버전에서와 같이 순차적으로 하나씩 입력하여 여러개의 장르를 복수 선택할 수 있다. 선택 및 선택 종료방식은 버전 선택과 동일하다. 
 8-4. 영상물 Runtime의 범위를 선택한다. 런타임의 x~y분 사이로 입력받는다. 
먼저 최소 러닝타임(x)을 입력받는다. 0이상의 정수만 입력받는다. 이후 최대 러닝타임(y)을 입력받는다.
0이상의 정수만 입력받으며 제한을두지 않으면 공백을 입력한다. 입력한 x분 이상 y분 이하의 runtime을 조건으로 선택한다. 
 조건입력이 모두 마무리 되면, 해당 조건에 부합하는 영상물의 식별번호화 제목을 나열한다. 
이후 정보 확인 및 평가 방식은 다른 메뉴와 동일하다. 

9. 모든 평가 내역 보기(필수 기능 3-C)
관리자는 모든 평가 내역을 볼 수있다. 이 때 제공되는 정보 영상물의 제목, 평균 평점, 유저의 아이디와 해당유저가 내린 평점이다.

10. 새로운 영상 올리기 기능(필수 기능 4-A)
관리자는 새로운 영상을 추가할 수 있다. 이 때 '*'가 붙은 입력사항은 필수적으로 입력해야하고 입력하지 않을 시 다음 단계로 넘어가지 않는다. 또한 각각의 제약사항을 지켜야 다음 단계로 넘어갈 수 있다. 모든 정보를 입력 후 '영상물 추가 완료'가 출력되면 DB에 정상적으로 저장된 것이다.

11. 기존 영상 수정 기능(필수 기능 4-B)
수정할 영상의 이름을 입력한다.
DB에서 해당 이름이 포함된 모든 영상물을 사용자에게 보여준다.
없는 경우 다른 영상을 찾을 것인지 물어본다.
해당 이름이 포함된 영상물을 보여주고, 선택할 영상물의 번호를 입력한다.
수정을 원하는 항목에 대해 입력하고 원하지 않는 항목은 엔터를 통해 넘어간다.
입력한 정보를 바탕으로 DB에 있는 튜플을 업데이트한다.
다른 영상도 수정할 것인지 물어보고 원한다면 다시 기능을 시작하고
원하지 않으면 이전 메뉴로 돌아간다.

12. 회원 등급 변경 기능
모든 회원은 가입시 멤버쉽이 Basic으로 시작하는데,
관리자는 회원의 등급을 변경 시킬 수 있다.
변경할 회원의 ID를 입력한다.
회원이 존재하지 않는 경우 다시 입력받고, 존재하는 경우 변경할 등급을 목록에서 선택한다.
해당하는 회원의 ID의 멤버쉽 등급을 변경한다.

프로그램을 종료한다.
